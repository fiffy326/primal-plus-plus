## Installation

```sh
git clone https://github.com/fiffy326/primal.cpp
cd primal.cpp
make
sudo make install
```

```sh
# Other compilation options:
make gcc
make gcc-static
make clang
make clang-static
```
## Dependencies

Compilation is done with gcc by default.

Hyperfine is also required to use the `make benchmark` command.
